React éuma biblioteca JS para Front-end (criação de interfaces);
Single Page Application (Páginas com sem mudanças de contexto);
Utilizamos JSX para ser renderizado com HTML;
É preciso instalá-lo na máquina (create-react-app);
JSX é como um HTML, porém dentro de um code JS 
e podemos interpolar variáveis;
JSX é a principal maneira de escrever HTML com o React;
Props são valores passados para componentes, deixando dinâmicos;
O CSS no React pode ser adicionado de forma global, porém é possível 
estilizar cada componente;
Utilizamos o CSS Modules, basta criar Componente.module.css;
React Fragments permite criar um componente sem elemento pai;
Descomplica os nós do DOM;
Sintaxe é <> </>, sem nome para tag e criamos diretamente no JSX;
Eventos no React são os mesmos eventos do DOM;
O evento é atrelado a uma tag que irá executá-lo;
useState é um hook do React, conseguimos manusear o estado,
funciona bem com eventos, 
podemos atrelar a um evento com mudança de state;
Métodos podem ser ativados com componente filho do seu ancestral;
Sintaxe é a mesma coisa coisa do props de dados: props.meuEvento;
IF é uma renderização por condição, envolvemos as tags em {}, 
é possível usar o State para criar as condições;
Para renderizar uma lista precisa de um Array, 
depois utilizamos a função Map, para percorrer cada item da lista,
é possível unir operadores condicionais com a renderização de listas;
State Lift é uma técnica utilizada para compartilhar o state,
dependem do mesmo estado e definimos quem usa quem(setState).
React Router é um pacote para mudança de URLs da aplicação,
Podemos assim acessar outras views, sem o page reload,
trocando uma parte da aplicação,
precisa instalar este pacote no projeto,
e também realizar algumas mudanças em como o App é estruturado;
React Icons é um pacote de ícones externo,
precisamos adicionar ao projeto atráves do npm,
permite adicionar ícones com sintaxe parecida de componetes.
