React éuma biblioteca JS para Front-end (criação de interfaces);
Single Page Application (Páginas com sem mudanças de contexto);
Utilizamos JSX para ser renderizado com HTML;
É preciso instalá-lo na máquina (create-react-app);
JSX é como um HTML, porém dentro de um code JS 
e podemos interpolar variáveis;
JSX é a principal maneira de escrever HTML com o React;
Props são valores passados para componentes, deixando dinâmicos;
O CSS no React pode ser adicionado de forma global, porém é possível 
estilizar cada componente;
Utilizamos o CSS Modules, basta criar Componente.module.css.
